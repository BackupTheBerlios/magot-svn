Magot project

A cross-platform personal finance manager written in Python.

News :

Magot 0.1 has been released. It allows to :
  • Drag&Drop Accounts in the hierarchy.
  • Modify all fields of an Account, a Transaction or an Entry.
  • Change the opposite Account of an Entry.
  • Jump between Accounts following each Entry of a Transaction.
  • Sort Entries of an Account by clicking on any column headers.
For details, look at the history file.

Features :

Functional :

  • Double-entry book-keeping, respecting the equation : assets + expenses = liabilities + equity + revenue.
  • Hierarchy of drag-and-droppable Accounts that compute their own balance as the sum of nested balances.
  • Each Account holds Transactions.
  • Each Transactions holds Entries that link several Accounts for this Transaction.

Technical :

  • Define a Business Domain Model, implemented with the framework PEAK.
  • Use wxPython as the cross-platform GUI toolkit. Tested on Linux and Windows.
  • Use agile technics like YAGNI, DRY, Test-Driven and Domain-Driven development, OOP.
  • Object Persistence is done with Python Pickling.
  • Saves are transactionable with EditableFile.
  • Basic command lines.

Installation :

You have to use exactly WxPython 2.6.0.1 and at least Python 2.4.

After that, download a Magot archive and unzip it in a directory.
Modify the file bin/magot.bat and change the line set PYTHON=C:\tools\python241\python in order to use your own Python interpreter.

Then, you can launch the Magot GUI by clicking on magot.bat.
